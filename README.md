## 📱 Overview

**Cart Corner** is a fully responsive e-commerce mobile application built with Flutter, focused on delivering a clean and intuitive shopping experience. The project showcases polished UI design, smooth navigation, and a well-structured component-based layout tailored for modern online shopping apps.

---

## ✨ Features

- 🏠 **Home Screen** — Product browsing with categories and featured items
- 🛍️ **Product Listing** — Clean grid/list view with product cards
- 🛒 **Cart Management** — Add, remove, and review cart items
- 📦 **Product Details** — Detailed view with images, description, and pricing
- 📐 **Responsive Design** — Adapts seamlessly across different screen sizes and devices

## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed ([Flutter docs](https://docs.flutter.dev/get-started/install))
- Android Studio or VS Code with Flutter plugin

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/salahsaleh1015/cart-corner-app.git
cd cart-corner-app

# 2. Install dependencies
flutter pub get

# 3. Run the app
flutter run
```
